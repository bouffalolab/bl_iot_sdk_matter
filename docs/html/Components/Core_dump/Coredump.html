Core dump¶

在系统异常时dump出指定的内存数据

功能¶

  1. 可选配置默认dump区域，在exception时自动dump至UART，通过串口工具保存

  2. 输出端口可配（用户自行适配）：

  • 串口

  • flash

  • spi

  1. base64加密编码输出

  • 一次命令仅可包含一对``addr/len``

  • 若未配置默认dump区域，d命令不会响应；传入d命令时，其他指令会被忽略
